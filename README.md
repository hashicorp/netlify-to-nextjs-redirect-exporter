# netlify-to-nextjs-redirect-exporter

A library to convert a Netlify `_redirects` file into compatible Next.js rules

**Install dependencies**

```shell
npm i
```

## Active Development

```shell
npm run watch
```

## Run tests

```shell
npm t
```
